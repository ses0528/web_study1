자주 사용되는 HTML 태그 리스트(list of frequently used tags)


!doctype html : 관용적으로 코드 최상단에 표기, 문서 종류를 표시함 html : head와 body를 감싸는 태그. head 태그 위에 표기
head : 본문을 설명하는 윗부분은 head 태그로 묶기로 약속
body : 본문인 아랫부분은 body 태그로 묶기로 약속
title : 웹페이지 제목 울타리
meta : 웹 브라우져가 읽을 수 있도록 문자의 저장방식 알려주기, utf-8 등
div
a : anchor닻, HyperText link 기능

link

span
p : 단락 울타리
li : list, 목차 각 줄의 울타리, 줄바꿈
ul : unordered list, 목차 전체 줄의 울타리, 그룹핑
ol : ordered list, 목차 전체 줄의 울타리, 각 줄에 자동으로 순번 붙이기, 그룹핑
style
br : break, 줄바꿈, 닫는 태그 불필요
h1~h6 : heading, h1~h6, 제목 울타리, 글자 크기를 크게
